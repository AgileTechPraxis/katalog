## Anagrams

Write a program to generate all potential anagrams of an input string.

>
> Example:
>
> *Input* : `biro`
>
> *Output* :
> * `biro` `bior` `brio` `broi` `boir` `bori`
> * `ibro` `ibor` `irbo` `irob` `iobr` `iorb`
> * `rbio` `rboi` `ribo` `riob` `roib` `robi`
> * `obir` `obri` `oibr` `oirb` `orbi` `orib`
>
